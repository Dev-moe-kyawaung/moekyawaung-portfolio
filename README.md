# Moe Kyaw Aung | Supreme Portfolio

Welcome to my ultimate portfolio website! This “Supreme Ultra Pro Max” package showcases my skills, projects, and provides a modern, responsive design with smooth animations.

## Features

- Fully responsive design
- Hero section with Gravatar avatar
- About section with skills & personal description
- Project section with hover animations
- Contact form
- Smooth scrolling navigation
- Footer with social links
- Premium UI/UX styling

## Technologies Used

- HTML5 & CSS3
- JavaScript (for smooth scrolling & interactivity)
- Google Fonts (Roboto)
- Optional: Can be adapted to Bootstrap, React, Next.js, Angular, Tailwind, Gatsby, Flutter, Kotlin Ktor, Ruby Sinatra
